# Vue.js Hello World

這是一個使用Vue.js建立的簡單Hello World網站。

## 功能

- 顯示問候訊息
- 點擊按鈕可以切換不同的訊息

## 技術

- Vue.js 3 (使用Composition API)
- HTML
- CSS
- JavaScript

## 如何運行

只需在瀏覽器中打開`index.html`文件即可查看網站。
